# Gitscord
**Gitscord** is a Free and Open-Source Discord bridge for GitHub. It allows you to sync your Discord issues from various repos to a Discord forum channel.

# Features
- Link your Discord and GitHub account
- Allow Discord users to open issues and send comments **via Discord**, even without an account.
- Support almost every issue events and display them on Discord.
- Bridge Tags/Labels and Title

# Building
This project requieres maven. To get a fat jar, run
```sh
mvn clean package
```
